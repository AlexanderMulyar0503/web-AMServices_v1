# AMServices

Различные сервисы и игры, доступные из локальной сети


## Возможности

### Сервисы
- Генератор паролей
- Разложение чисел на мнодители
- Генератор случайных чисел
- RGB-палитра

### Игры
- Крестики-нолики
- Игра в 15 (Пятнашки)


## Системные требования
- Веб-сервер (Apache, Nginx, Lighttpd)
